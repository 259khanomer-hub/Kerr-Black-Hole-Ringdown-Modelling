# Ringdown Templates in GR and Modified Gravity

This repository contains modelling tools and example notebooks for constructing and studying **black-hole ringdown waveform templates** in **General Relativity (GR)** and **modified-gravity scenarios**.

## Features

- Multimode Kerr ringdown template construction  
- Phenomenological mode-dependent deviations from GR  
- Example theory-inspired modified-gravity ringdowns  
- Waveform comparisons and mismatch studies  
- Simple injections and detectability demonstrations  

## Motivation

The ringdown phase encodes the quasinormal mode spectrum of the remnant black hole.  
Deviations in mode frequencies or damping times can provide tests of gravitational physics beyond GR.  

This repository provides a lightweight research workflow for exploring such deviations at the waveform level.
